# data-science-internship-02
## Overview
This repository contains solutions for **Logic Building Task – 2**.  
The task focuses on improving Python fundamentals, logic building, and basic data structure usage.

All problems are implemented in **Python using Jupyter Notebook**.

## Problems Covered
1. Unique Words in a Sentence  
2. Highest Salary from Employee Data  
3. Find Maximum and Minimum Values  
4. Count Products Above Price Threshold  
5. Calculate Attendance Percentage  
6. Remove Duplicate Phone Numbers  
7. Count Character Frequency  
8. Convert List to Tuple  
9. Check if Key Exists in Dictionary  
10. Calculate Average Marks  

## Technologies Used
- Python  
- Jupyter Notebook / Google Colab  
- GitHub  

## File Included
- `logic_building_task2.ipynb` → Contains all solutions  

## How to Run
1. Open the notebook in **Jupyter Notebook / Google Colab**  
2. Run all cells sequentially  
3. Check outputs for each problem
